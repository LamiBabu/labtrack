# LabTrack - Lab Asset Management System

## 📌 Description
A web-based application to manage lab equipment, students, and issue records.

## 🚀 Features
- Admin Dashboard
- Student Dashboard
- Issue & Return Equipment
- Complaint System
- MySQL Database with Constraints

## 🛠️ Technologies Used
- Python (Flask)
- MySQL
- HTML, CSS, JavaScript

## 🧠 Concepts Used
- DDL & DML
- JOIN Queries
- Aggregate Functions
- Views
- Triggers
- Constraints

## ▶️ How to Run
1. Start MySQL and import the database
2. Run the application:
   python backend/app.py
3. Open browser and go to:
   http://localhost:5000
